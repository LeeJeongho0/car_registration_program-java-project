# car_registration_program-java-project

##  차량등록 프로그램 소개
- 입주민 차량과 방문객 차량의 정보를 입력 후, DataBase(MySql)과 연동하여 데이터를 저장한다.

| Language | Java 17.0.6 |
| Editor | Eclipse IDE 2022-06 (4.24.0) |
| DBMS | MySQL Workbench 8.0.17 |
